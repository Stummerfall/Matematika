Теоремы

Теоремы исключения констант:
x ∨ 1 = 1;
x ∨ 0 = 0;
x ∧ 1 = x;
x ∧ 0 = 0.


Теоремы идемпотентности (тавтологии, повторения):
x ∨ x = x;
x ∧ x = x.
Для n переменных:
x ∨ x ∨…∨ x = x;
x ∧ x ∧…∧ x = x.


Теорема противоречия:
x ∧ ¬ x = 0.


Теорема "исключённого третьего:"
x ∨ ¬ x = 1.


Теорема двойного отрицания (инволюции):
¬ (¬ x) = x.
