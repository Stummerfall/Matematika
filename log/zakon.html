Законы логики

Для логических операций верны законы алгебры логики, которые можно использовать для упрощения логических выражений. При этом следует отметить, что в логике высказываний отвлекаются от смыслового содержания высказывания и ограничиваются рассмотрением его с той позиции, что оно либо истинно, либо ложно.

Ассоциативный (сочетательный) закон:
x1 ∨ (x2 ∨ x3 ) = (x1 ∨ x2 ) ∨ x3
x1 ∧ (x2 ∧ x3 ) = (x1 ∧ x2 ) ∧ x3.


Коммутативный (переместительный) закон:
x1 ∨ x2 = x2 ∨ x1;
x1 ∧ x2 = x2 ∧ x1.


Дистибутивный (распределительный) закон:
(x1 ∨ x2) ∧ x3 = (x1 ∧ x3) ∨ (x2 ∧ x3);
(x1 ∧ x2) ∨ x3 = (x1 ∨ x3) ∧ (x2 ∨ x3).


Законы де Моргана (законы общей инверсии или дуальности):
¬ x1 ∨ ¬ x2 = ¬ (x1 ∧ x2);
x1 ∨ x2=¬ (¬ x1 ∧ ¬ x2);
¬ x1 ∧ ¬ x2 = ¬ (x1 ∨ x2);
x1 ∧ x2=¬ (¬ x1 ∨ ¬ x2).


Закон поглощения (элиминации):
x1 ∨ (x1 ∧ x2) = x1;
x1 ∧ (x1 ∨ x2 ) = x1.


Закон склеивания (исключения):
(x1 ∧ x2) ∨ (x1 ∧ ¬ x2) = x1;
(x1 ∨ x2) ∧ (x1 ∨ ¬ x2) = x1.
